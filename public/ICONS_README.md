# PWAアイコンについて

このディレクトリには、PWA（Progressive Web App）用のアイコンファイルを配置してください。

## 必要なアイコン

### 1. `icon-192x192.png`
- サイズ: 192x192 ピクセル
- 用途: ホーム画面アイコン、スプラッシュ画面

### 2. `icon-512x512.png`
- サイズ: 512x512 ピクセル
- 用途: 高解像度ディスプレイ用アイコン

### 3. `apple-touch-icon.png` (オプション)
- サイズ: 180x180 ピクセル
- 用途: iOSデバイスのホーム画面アイコン

### 4. `favicon.ico` (オプション)
- サイズ: 32x32 または 16x16 ピクセル
- 用途: ブラウザタブアイコン

## アイコンのデザインガイドライン

### カラー
- メインカラー: `#f59e0b` (アンバー)
- 背景カラー: `#fffbeb` (アンバー50)

### デザインコンセプト
- 子供向けで親しみやすいデザイン
- シンプルで認識しやすい
- プログラミング要素（ブロック、コードなど）を含む

### 推奨デザイン案
1. **ブロックアイコン**
   - カラフルなブロックが積み重なったデザイン
   - Scratchのようなビジュアル

2. **コードシンボル**
   - `</>` や `{}` などのシンボル
   - 丸みを帯びた子供向けフォント

3. **キャラクターアイコン**
   - かわいいロボットやマスコット
   - 笑顔で親しみやすい表情

## アイコン生成ツール

### オンラインツール
- [Favicon Generator](https://realfavicongenerator.net/)
- [PWA Asset Generator](https://www.pwabuilder.com/)
- [Canva](https://www.canva.com/) - デザイン作成

### コマンドラインツール
```bash
# 一つの大きな画像から複数サイズを生成
npm install -g pwa-asset-generator
pwa-asset-generator logo.svg public --icon-only
```
